Política de Segurança de Dados Segurança do Data Center O alexti.net.br é hospedado no GitHub, onde muitas medidas estão em vigor para proteger a segurança O acesso físico ao site de dados é altamente restrito, registrado e monitorado A autenticação multifator é necessária para entrar no data center Proteção contra perda de dados O alexti.net.br realiza backups de dados duas vezes ao dia em um site de dados em uma região separada Os exercícios de recuperação de desastres são realizados trimestralmente Cada cliente tem um banco de dados segregado para evitar corrupção e sobreposição Segurança em nível de aplicativo As senhas das contas de usuário são criptografadas e exigem letras minúsculas, letras maiúsculas, um número e um caractere especial Todas as solicitações de aplicativos são criptografadas usando TLS Todos os dados são criptografados em repouso Realizamos testes OWASP semanais Segurança e políticas internas de TI Todos os funcionários e contratados devem passar por uma verificação de antecedentes Todos os funcionários e contratados com acesso a dados confidenciais são obrigados a concluir o treinamento de segurança e FERPA anualmente. O acesso a dados sensíveis é limitado conforme a necessidade. Funcionários e contratados estão proibidos de baixar dados sensíveis, a menos que estejam relacionados a uma solicitação do cliente, e são treinados sobre como criptografar esses dados. Conformidade com SOC O CourseArc passa anualmente por uma auditoria externa SOC 2 Tipo 2 Divulgação Responsável Se você descobrir uma vulnerabilidade no aplicativo CourseArc, entre em contato com support@coursearc.com , o que garantirá que a Equipe de Resposta a Violações de Dados seja notificada e investigada minuciosamente. Notificação de violação de dados Se tivermos motivos para acreditar que uma parte não autorizada obteve acesso a quaisquer dados não criptografados que não sejam informações publicamente disponíveis, enviaremos uma notificação sobre a violação. Entraremos em contato com o(s) principal(is) ponto(s) de contato designado(s) pelo cliente por telefone e e-mail em até 48 horas.
